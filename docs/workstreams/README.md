# Workstreams

Workstream files are executable task contracts shared by LizBing, Codex, and
Claude. Copy [`TEMPLATE.md`](./TEMPLATE.md), give the file a stable milestone
name such as `R2-ARCHITECTURE-GATES.md`, and link the active one from
[`../PROJECT_STATUS.md`](../PROJECT_STATUS.md).

There is at most one implementation owner and one reviewer per workstream.
Closed contracts remain in this directory as an audit trail; do not rewrite
their original scope after completion. Corrections should be clearly dated.
